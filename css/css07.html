목록 속성

list-style : type position image

메뉴나 목록의 기호를 표시하는 속성



목록 기호 - 순서가 없는 거

list-style-type : disc | circle | square
decimal | armenian | georgian | hebrew | hiragana
katakana | lower-alpha | lower-greek |lower-latin
lower-roman | upper-alpha | upper-greek | upper-latin
upper-roman | cjk-ideographic

li 요소에 표시되는 목록 기호로 부모 요소가 ol인 경우는 숫자가 순서대로 앞에 표기되고, ul인 경우는 disc가 앞에 표시된다.



목록 기호 위치

list-style-position : inside | outside



목록 이미지

list-style-image : none | url(파일 경로 및 이름)

목록 기호에 들어갈 자리에 이미지가 표시되며, 파일경로나 이름은 ""(큰따옴표), ''(작은따옴표)로 묶을 수 있으며, 생략 가능하다.
다만, 중간에 띄워쓰기나 한글 및 특수문자로 이루어져 있는 경우는 반드시 따옴표로 묶어야 한다.



마우스 포인터

해당 요소의 영역에 마우스 포인터가 위치되면, 그 마우스 포인터 모양을 지정할 수 있는 속성



그림자 속성

shadow : text-shadow, box-shadow

x텍스트나 박스의 외부 그림자 또는 내부 그림자를 말하며, 크기 속성에 포함되지 않는다.



텍스트 그림자

text-shadow : x-length y-length blur-length color

x를 양의 값으로 입력하면, 그림자의 방향은 오른쪽에 발생되며, 음의 값으로 입력하면, 그림자의 방향은 왼쪽에 발생됨.
y를 양의 값으로 입력하면, 그림자의 방향은 아래쪽에 발생되며, 음의 값으로 입력하면 그림자의 방향은 위쪽에 발생된다.
x y blur color 양의 여러 차례에 걸쳐 지정하면, 입체 모양의 그림자가 표시된다.



박스 그림자

box-shadow : x-length y-length blur-length color

box-shadow : x-length y-length blur-length color spread-length color

x를 양의 값으로 입력하면, 그림자의 방향은 오른쪽에 발생되며, 음의 값으로 입력하면, 그림자의 방향은 왼쪽에 발생됨.
y를 양의 값으로 입력하면, 그림자의 방향은 아래쪽에 발생되며, 음의 값으로 입력하면 그림자의 방향은 위쪽에 발생된다.
x y blur color 양의 여러 차례에 걸쳐 지정하면, 입체 모양의 그림자가 표시된다.

box-shadow나 text-shadow의 그림자의 양은 px또는 pt 단위로 기재한다.
