글자/문단 관련 속성

중요한 것들은 볼드체 쳐놨음

글자 관련 : color, font-size, font-weight, font-stretch(장평), text-transform(대, 소문자 바꿔주는 역할), font-varient(소문자의 크기처럼 아예 줄임), font-style(기울어지게), font-family(글꼴), font-kerning(자간, 커서가 놓여있을 때 적용), @font-face

문단 관련 : letter-spacing(글자사이의 간격), line-height, word-spacing(단어 사이의 간격) text-align, text-indent(들여쓰기), text-decoration, text-justify, text-orientation, text-overflow, white-space, word-break



글자색

color : 웹컬러명 | 16진 코드 | rgb()/rgba() | hs1()/hsla()



글자 크기

font-size : keyword | length

keyword : xx-small | smaller | small | medium | large | x-large | xx-large

length unit : px, em(rem), pt 등

keyword 를 사용하여 글자크기를 지정할 경우는 브라우저 마다 모두 다르므로, length 값으로 지정하는 것이 좋다.



글자 두께

font-weight : keyword | number(단위가 붙지 않음)

keyword : border | bold | normal | lighter

number : 100 ~ 900로 100단위로 기재. 100이 가장 가늘고, 900이 가장 두껍다.



글자 장평

font-stretch : ultra-condensed(가늘다) | extra-condensed | condensed | semi-condensed | normal | semi-expanded | expanded | extra-expanded | ultra-expanded

장평(font-stretch)이란? 한 글자의 가로크기와 세로크기의 비율이며, 기준은 세로크기에 비해 가로크기가 얼마나 큰지에 대하여 값을 부여한다.



글자 스타일

font-style : normal | italic | oblique |

기울임꼴의 유무를 나타내는 속성



글자 변형

font-varient : normal | small-caps

영문의 경우 대문자와 소문자의 크기가 각각 달라서 균일하게 글자크기나 폭을 지정하기 힘들므로 대문자를 소문자 크기로 조절하기 위한 속성



글자 변환

text-transform : none | uppercase | lowercase | capitalize

해당 단어나 문장의 영문을 대문자 또는 소문자로 변환할 때 사용



글꼴 지정

font-family : 1차폰트, [, 2차폰트][, 3차폰트]

서버에 설치되어 있는 폰트의 이름을 지정하며, 만약 폰트의 이름 중간에 공백이 있거나 아니면 한글과 같은 영문이 아닌 경우는 반드시 인용부호인 ""나 ''로 감싸서 표기해야하며, 만약, 1차 폰트가 존재하지 않는 경우는 2차 폰트가 적용되고, 여기서 2차 폰트도 적용될 수 없다면, 3차 폰트가 적용하게 된다. 다만, 3차 폰트가 적용되지 못하는 경으를 대비하여, 3차 폰트는 브라우저 기본적으로 제공하는 폰트를 지정하여야 함.

브라우저 제공하는 기본 폰트는 해당 서버나 클라이언트에 설치가 되어있지 않아도 자공 제공되는데, sans-serif(고딕계열), serif(명조 계열), cursive(궁서 계열), monospace(가로와 세로의 장평이 같은 폰트), fantasy(디자인 계열) 등을 말함.



커서 위치의 자간

font-kerning : auto | normal | none



자간

letter-spacing : normal | length

length : px, em, pt 등의 단위를 주로 활용

해당 요소 안에 글자와 글자 사이의 간격



어간

word-spacing : normal | length

length : px, em, pt 등의 단위를 주로 활용

단어와 단어 사이의 간격



행 높이

line-height : normal | number | length

값으로 숫자만 기재할 경우는 em단위처럼 현재 문단의 글자 크기의 배수가 되며, 숫자 뒤에 단위가 들어가는 경우는 해당하는 단위의 크기가 적용됨.



문단 정렬

text-align : left | right | center | justify

문단의 가로방향에 대한 정렬



들여쓰기/내어쓰기

text-indent : length

length 단위 : px, %, pt 등

들여쓰기 : 다른 줄에 비해 첫번째 줄이 더 들어가게 쓰는 것
text-indent:20px;

내어쓰기 : 다른 줄에 비해 첫번쨰 줄이 더 나오게 쓰는 것
text-indent:-20px;



글자 장식

text-decoration : none | underline | underline | overline | line-through

글자의 위, 중간, 아래에 줄을 표시하도록 하는 속성

none을 제외한 다른 값은 중복해서 사용할 수 있다.
text-decoration:overline underline
text-decoration:overline underline double #369, overline wave #c00;

세부 속성

text-decoration-color : 줄의 색

text-decoration-line : 줄의 위치(underline/overline/line-through)

text-decoration-style : 줄의 종류(solid/wave/double/dotted/dashed)



배분 정렬

text-justify : auto | none | inter-character | inter-word

단어 또는 글자 단위로 한 줄의 데이터를 분량을 똑같이 배분정렬 할 수 있는 속성



글자 진행 방향

text-orientation : mixed | upright | sideways | sideways-right| use-glyph-orientation



글자 흘러 넘침

text-overflow : clip | ellipsis | 특정문자

text-overflow는 overflow:hidden이며, white-space:nowrap 인 경우에 유효하다.



내용 보호/줄 끝 처리

white-space : normal | nowrap(보호x, 자동줄바꿈x) | pre(자동 줄바꿈X) | pre-line(자동 줄바꿈O) | pre-wrap(줄바꿈, 끝에 엔터 친것도 밑으로 내려보냄)

하나의 단어가 한 줄에 입력되어야만 하기 때문에 그 때 한 줄에 단어가 배치가 되는지, 아니면, 영문 배치 문법을 무시하고, 단어 중간을 자를지 또는 하나의 문장을 한 줄로만 표시할지 그 문장의 줄의 끝을 어떻게 처리할지 정하는 속성



단어 처리

word-break : normal | break-all | keep-all | break-word

한 줄 또는 한 문장 안에서의 단어의 끝 처리 부분을 어떻게 할지 정하는 속성
